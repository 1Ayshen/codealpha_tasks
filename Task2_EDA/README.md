# Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis on the Books to Scrape dataset collected during the web scraping task.

## Dataset Information

* Total records: 1000 books
* Columns: Title, Price, Rating, Page

## Analysis Performed

* Dataset overview
* Data type inspection
* Missing value analysis
* Duplicate record analysis
* Statistical summary
* Rating distribution analysis
* Most expensive and cheapest book identification
* Average, median, and standard deviation of book prices
* Average price by rating
* Average price by page

## Key Findings

* The dataset contains no missing values.
* No duplicate records were found.
* The average book price is approximately £35.07.
* The median price is £35.98.
* The most expensive book costs £59.99.
* The cheapest book costs £10.00.
* Books with a Five-star rating have the highest average price among all rating categories.
