# Data Visualization with Matplotlib
This project visualizes the Books to Scrape dataset using Matplotlib.
## Visualizations
* Rating Distribution (Bar Chart)
* Price Distribution (Histogram)
* Book Price Box Plot
* Average Price by Rating
* Rating Distribution (Pie Chart)
* Top 10 Most Expensive Books
* Top 10 Cheapest Books
* Price vs Website Page (Scatter Plot)
## Tools Used
* Python
* Pandas
* Matplotlib
## Dataset
The dataset contains 1000 books collected from the Books to Scrape website during the web scraping task.
