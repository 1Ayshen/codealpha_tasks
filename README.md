# CodeAlpha Data Analytics Internship Tasks

This repository contains the tasks completed during my **CodeAlpha Data Analytics Internship**. The projects demonstrate skills in web scraping, exploratory data analysis (EDA), and data visualization using Python.

## Repository Structure

### Task 1 – Web Scraping

**Objective:** Collect book information from the Books to Scrape website.

**Tools Used:**

* Python
* Requests
* BeautifulSoup
* Pandas

**Features:**

* Scraped 1000 books
* Extracted book title, price, rating, and page number
* Saved the collected data into a CSV file

---

### Task 2 – Exploratory Data Analysis (EDA)

**Objective:** Analyze the scraped dataset to understand its structure and key insights.

**Analysis Performed:**

* Dataset overview
* Data type inspection
* Missing value analysis
* Duplicate record analysis
* Statistical summary
* Rating distribution
* Price analysis
* Average price by rating
* Most expensive and cheapest books

---

### Task 3 – Data Visualization

**Objective:** Visualize the dataset using Matplotlib.

**Visualizations Created:**

* Rating Distribution (Bar Chart)
* Price Distribution (Histogram)
* Book Price Box Plot
* Average Price by Rating
* Rating Distribution (Pie Chart)
* Top 10 Most Expensive Books
* Top 10 Cheapest Books
* Price vs Website Page (Scatter Plot)
