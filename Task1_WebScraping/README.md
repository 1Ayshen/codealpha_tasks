# Task 1 – Web Scraping

## Objective

The goal of this project is to collect book information from the **Books to Scrape** website using Python web scraping techniques.

## Website

Books to Scrape: https://books.toscrape.com/

## Tools Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Data Collected

The scraper extracts the following information for each book:

* Book Title
* Price
* Rating
* Website Page Number

A total of **1000 books** were collected and saved into a CSV file (`books.csv`).

## Project Files

* `web_scraping.py` – Python script used for web scraping
* `books.csv` – Scraped dataset containing 1000 books

## Learning Outcomes

Through this project, I learned how to:

* Send HTTP requests using the Requests library
* Parse HTML with BeautifulSoup
* Extract and organize website data
* Store scraped data in CSV format using Pandas
* Automate data collection from multiple web pages

## Conclusion

This project demonstrates the fundamentals of web scraping and serves as the foundation for the following tasks: Exploratory Data Analysis (EDA) and Data Visualization.
